Technical Overview

Wind Whisper Bot is a full-stack Conversational AI demo designed to demonstrate the technical and architectural skills required for an IVA Voice and Digital Chat Developer role.

This application features real-time communication with Kore.ai using live IVA workflows and a middleware integration via Railway to manage webhooks and dynamic response delivery. It emphasizes practical implementation of API orchestration, conversational design, and backend logic with clean UX principles.
